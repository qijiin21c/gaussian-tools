# Gaussian 计算技能选择流程（初学者指南）

## 总览

```
开始
  │
  ▼
┌─────────────────────────────────┐
│  Step 1: task-methods           │
│  判断计算任务类型                │
└──────────────┬──────────────────┘
               │
               ▼
    根据任务类型，选择下方对应流程
```

## 流程 1：DFT 密度泛函计算（标准路线）

```
task-methods → 密度泛函计算
  │
  ▼
┌──────────────────────────────┐
│  dft-functional              │
│  选择合适的泛函               │
│  (B3LYP, PBE0, M06-2X, ωB97X-D 等) │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│  basis-set                   │
│  选择合适的基组               │
│  (6-31G*, def2-SVP, cc-pVDZ 等)    │
└──────────────┬───────────────┘
               │
               ▼
     是否涉及弥散函数？
       ╱          ╲
     是            否
     │             │
     ▼             │
┌────────────────────┐      │
│  diffuse-functions │      │
│  选择弥散方案       │      │
└────────┬───────────┘      │
         │                  │
         └────────┬─────────┘
                  ▼
     是否需要考虑色散校正？
       ╱          ╲
     是            否
     │             │
     ▼             │
┌────────────────────┐      │
│  dft-d-dispersion  │      │
│  选择色散校正类型   │      │
│  (D3, D3BJ, D4 等) │      │
└────────┬───────────┘      │
         │                  │
         └────────┬─────────┘
                  ▼
           构建 .gjf 输入文件
                  │
                  ▼
           提交计算
                  │
                  ▼
     计算是否正常完成？
       ╱              ╲
     是              否（SCF不收敛）
     │                │
     │                ▼
     │      ┌──────────────────┐
     │      │  scf-convergence │
     │      │  解决SCF收敛问题  │
     │      │  (SCF=XQC, etc.) │
     │      └────────┬─────────┘
     │               │
     │               ▼
     │         重新提交计算
     │
     ▼
     几何优化是否收敛？
       ╱              ╲
     是              否
     │                │
     │                ▼
     │      ┌────────────────────┐
     │      │  opt-convergence   │
     │      │  解决几何收敛问题   │
     │      │  (opt=calcall,     │
     │      │   opt=gdiis, etc.) │
     │      └────────┬───────────┘
     │               │
     │               ▼
     │         重新提交计算
     │
     ▼

     │             ▼
     │   ┌────────────────────┐
     │   │  small-imag-freq   │
     │   │  处理小虚频         │

     │   └────────┬───────────┘
     │            │
     │            ▼
     │      是否大虚频？
     │        ╱      ╲

     │      │
     │      ▼
     │  ┌────────────────────┐
     │  │ eliminate-imag-freq│
     │  │  消除较大虚频       │
     │  │  (微扰结构重新优化) │
     │  └────────┬───────────┘
     │           │
     │           ▼
     │     重新提交计算
```

## 技能速查表

| 步骤 | 技能 | 用途 |
|------|------|------|
| 1 | `task-methods` | 判断计算任务类型（第一步） |
| 2 | `dft-functional` | 选择密度泛函 |
| 3 | `basis-set` | 选择基组 |
| 4 | `diffuse-functions` | 选择弥散函数（阴离子、Rydberg 态等） |
| 5 | `dft-d-dispersion` | 选择色散校正 |
| 6 | `large-system-weak-interaction` | 大体系弱相互作用 |
| 7 | `bsse-correction` | BSSE 校正（结合能计算） |
| 8 | `opt-convergence` | 几何优化收敛问题 |
| 9 | `restricted-opt` | 限制性优化 |
| 10 | `solvation-free-energy` | 溶剂化自由能 |
| 11 | `ionic-liquid-solvation` | 离子液体溶剂化 |
| 12 | `pes-scan` | 势能面扫描 |
| 13 | `basis-extrapolation` | 基组外推至 CBS 极限 |
| 14 | `tdfft-gaussian` | TDDFT 计算 |
| 15 | `excited-state` | 激发态计算 |
| 16 | `nmr-chemical-shift` | NMR 化学位移 |
| 17 | `metal-surface-cluster` | 金属表面团簇模型 |
| 18 | `lanthanide-complexes` | 镧系复合物 |
| 19 | `pseudopotential` | 赝势（重元素） |
| 20 | `eliminate-imag-freq` | 消除较大虚频 |
| 21 | `small-imag-freq` | 处理小虚频 |
| 22 | `irc` | IRC 计算（过渡态确认） |
| 23 | `scf-convergence` | SCF 收敛问题 |
